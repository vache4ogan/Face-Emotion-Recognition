# Facial Emotion Recognition (FER) с использованием CNN

Данный проект реализует сверточную нейронную сеть (CNN) для классификации эмоций человека по 7 категориям. Модель построена на базе **TensorFlow/Keras** и обучена на наборе данных **FER2013**.

## 📊 Результаты обучения
Модель демонстрирует уверенную обобщающую способность, учитывая сложность и "шумность" исходных данных:
* **Точность на тренировочных данных (Train Accuracy):** ~69%
* **Точность на валидации (Validation Accuracy):** **63%**
* **Классы:** Гнев (Angry), Отвращение (Disgust), Страх (Fear), Счастье (Happy), Нейтральное (Neutral), Грусть (Sad), Удивление (Surprise).

*Примечание: Для датасета FER2013 точность в диапазоне 60-70% является отличным результатом, так как изображения имеют низкое разрешение (48x48) и часто содержат ошибки в разметке.*

## 🧠 Архитектура нейросети
Сеть состоит из 4 последовательных сверточных блоков, оптимизированных для извлечения признаков мимики:

1. **Входной слой:** Изображения 48x48 пикселей в градациях серого (Grayscale).
2. **Аугментация данных:** В модель встроены слои `RandomFlip`, `RandomRotation` и `RandomZoom` для повышения устойчивости к разным ракурсам лиц.
3. **Сверточные блоки:**
    * Увеличение количества фильтров: **32 → 64 → 128 → 256**.
    * **Batch Normalization** после каждой свертки для стабилизации и ускорения обучения.
    * **Max Pooling** для уменьшения размерности и выделения главных признаков.
    * **Dropout (0.25 - 0.5)** для борьбы с переобучением (случайное отключение нейронов).
4. **Полносвязная голова (Dense):** Два плотных слоя (256 и 128 нейронов) и финальный слой с функцией **Softmax** для классификации по 7 классам.

## 🛠 Технологический стек
* **Язык:** Python 3.x
* **Библиотеки:** TensorFlow 2.x, Keras, NumPy, Pillow.
* **Инструменты обучения:**
    * `ModelCheckpoint`: Автоматическое сохранение только лучшей итерации модели (по `val_accuracy`).
    * `EarlyStopping`: Остановка обучения при прекращении улучшения лосса.
    * `GPU Support`: Настройка динамического роста памяти видеокарты (Memory Growth).
